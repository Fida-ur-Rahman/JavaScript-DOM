# JavaScript DOM Manipulation

A simple project demonstrating how to use JavaScript to interact with and manipulate the Document Object Model (DOM).

## Features
- Select and modify HTML elements
- Handle user events (click, input, etc.)
- Update content dynamically
- Beginner-friendly DOM examples

## Technologies Used
- HTML
- CSS
- JavaScript (ES6+)

## What is DOM?
The DOM (Document Object Model) represents the structure of a web page. JavaScript uses the DOM to access, change, add, or remove elements dynamically.
